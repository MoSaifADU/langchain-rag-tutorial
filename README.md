🤖 RAG Document Chatbot
A Retrieval-Augmented Generation (RAG) application that allows you to upload documents and have context-aware conversations. Hosted on Streamlit and powered by OpenAI. 


📂 Persistent Vector Store: Uses ChromaDB to index and store document embeddings for fast retrieval.
🧠 Intelligent RAG Pipeline: Built with LangChain to handle document loading, chunking, and retrieval logic.
🔍 Source Attribution: Every response includes direct citations from your uploaded files to eliminate hallucinations.
💬 Conversation Memory: Tracks chat history across the session for natural, multi-turn dialogues.
🚀 Modern UI: A sleek, responsive interface built with Streamlit. 
🛠️ Tech Stack
Language: Python 3.12+
Orchestration: LangChain
LLM & Embeddings: OpenAI (GPT-4o/GPT-3.5)
Vector Database: Chroma
Frontend: Streamlit
